Sebagaimana namanya, ibislinux mempunyai sejarah nama dimana diambil dari nama ibis, ibis yaitu nama latin dari burung hantu. Mengapa si yang di ambil burung hantu, bukan burung garuda maupun burung elang yang notabene lebih gagah?

Burung Hantu merupakan simbol ilmu pengetahuan, kami menggunakan simbol burung hantu (ibis) karena kami semua memang masih belajar, kami semua haus akan ilmu pengetahuan yang memang tidak ada batasnya selain tuhan.

IbisLinux adalah distribusi GNU / Linux berdasarkan Linux From Scratch, hasil karya dari sebuah ketertarikan pada open-source, dimana pengguna bisa mempelajari dari awal tentang aplikasi yang ada di dalamnya. Dengan penggunaan yang mudah serta ringan, IbisLinux lebih dikhususkan untuk device x64 Low End, karena keringanan nya, RAM 2GB pun bisa menjalankan Ibis Linux.

Pertanyaan yang umum atau sering kami temukan:

  1. Turunan apa ibislinux?
  2. Distro apakah ini?
  3. Kok hanya console, doang ndak ada GUI nya?
  4. Belum bisa di install ya?

Pertanyaan tersebut akan kami jawab:

  1. Turunan apa ibislinux?

    2. ibislinux adalah distro yang di buat dengan basic linuxfromscratch dimana semua aplikasi di kompilasi secama manual tanpa menggunakan paket manajer. Apakah selanjutnya ibislinux akan menggunakan paket manager? kami dari developer rencananya akan menggunakan scratchpkg.

  2. Distro apakah ini?

    3. ibislinux adalah simple penetration distro atau yang sering dikenal distro yang di gunakan untuk kegiatan penetration testing, bisa juga di gunakan untuk penggunaan sehari hari

  3. Kok hanya console, doang ndak ada GUI nya?

    4. untuk masuk ke mode GUI bisa, gunakan perintah startx di sana kalian bisa merasakan keringanan dari ibislinux

  4. Belum bisa di install ya?

    5. Saat ini belum bisa di install, masih dalam mode live cd, misalkan memang mau di install harus menggunakan rsync
